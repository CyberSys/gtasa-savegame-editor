  • Some known values of Version:
    • 0x35da8175 - Version 1.00 Unmodified EXE
    • 0x65f3e583 - Version 1.00 Modified EXE
    • 0x9a6ebe58 - Version 1.01 Unmodified EXE
    • 0x9345765e - Version 1.01 Modified EXE
    • 0xfd148df6 - Version 2.00 Unmodified EXE
    • 0x5d31cc22 - Version 2.00 (German)
    • 0xfd148df6 - PS2 Version 2 (Greatest Hits)
    • 0x641ddc4c - PS2 Version 1 (Original Edition)
  • Title is at most 24 characters long.
    Supported characters: ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz012345789!"$%&'()+,-/.:;<=>?\]
    ] > *, ( > [, ) > ]
  • Minute length is the length of a game minute in ms. (default 1000)
  • Weather timer and Global timer are in milliseconds
  • H, M and Dy are the same as Hour, Minute and Day of week
  • Vehicle camera: 0 = First person, 1 = Close, 2 = Normal, 3 = Far, 5 = Cinematic
  • Person camera: 1 = Close, 2 = Normal, 3 = Far
  • Ch is true if you used any cheats.
  • In is true when saved inside.
  • Inside is 1.0 when saved inside, 0.0 when saved outside.
  • Ri is true when there are riots in Los Santos.
  • Day of week is the day of the week in the game, starting with 1 on Sunday.
    The first day in the game is Wednesday.
  • Day w is the day of the week of your computer, starting with 0 on Sunday.
  • The last 8 shorts represent the system time, not the game time.
  • Target is a blip link (see block 9) to the target blip on the radar.
    These bytes are all 00 if there is no target set.
  • TN is all taxi's have nitro
  • P is prostitutes pay you
